# Quality Gate Evidence

Dieses Verzeichnis ist für nachvollziehbare Qualitätsnachweise vorgesehen.

Beispiele:
- zusammenfassende Gate-Reports
- dokumentierte manuelle Checks
- spätere Exportdateien aus Prüfscripten

Generierte oder maschinell erzeugte Artefakte sollen später unter
`docs/_ops/quality_gates/generated/` abgelegt werden.
