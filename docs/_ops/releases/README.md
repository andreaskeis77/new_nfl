# Release Evidence

Dieses Verzeichnis ist für menschlich lesbare Release- oder RC-Evidence vorgesehen.

Beispiele:
- kurze Release-Zusammenfassung
- Scope
- Gate-Ergebnisse
- bekannte Restrisiken
- Verweise auf Handoff und PROJECT_STATE

Generierte oder maschinell erzeugte Artefakte sollen später unter
`docs/_ops/releases/generated/` abgelegt werden.
