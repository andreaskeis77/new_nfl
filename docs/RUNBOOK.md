# NEW NFL – Runbook v0.1

## 1. Zweck

Dieses Dokument wird das betriebliche Nachschlagewerk für Start, Stop, Diagnose,
Recovery und späteren VPS-Betrieb.

## 2. Aktueller Stand

Noch keine Runtime und noch keine produktionsnahen Dienste vorhanden.

## 3. Später hier erwartet

- Prozess- und Dienstübersicht
- Ports / Bindings / Endpunkte
- Start-/Stop-Befehle
- Logpfade
- Scheduled Tasks / Services
- Health-Checks
- Recovery-Schritte
- bekannte Störbilder
- Unterschiede DEV-LAPTOP vs. VPS-USER vs. VPS-ADMIN

## 4. Regel

Vor dem ersten produktionsnahen Runtime-Release muss dieses Dokument
von Platzhalter auf betriebstaugliches Niveau gehoben werden.
